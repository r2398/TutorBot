
# TutorAnna - Multilingual AI Tutor for Grades 6–12

## Overview
TutorAnna is a comprehensive, voice-first AI tutor designed to provide personalized academic support for students in Grades 6–12 across Mathematics, Science, and Social Science.

## Key Features Implemented

### 1. **Personalized Onboarding**
- Student name and APAAR ID collection
- Grade selection (6-12)
- Multilingual support (22 Indian languages)
- Preferred subject selection

### 2. **Voice-First Multilingual Interaction**
- Web Speech API integration for voice input/output
- Support for 22 Indian languages
- Real-time transcript display
- Text-to-speech responses
- Bilingual and code-mixed usage support

### 3. **Multimodal Input Support**
- **Voice Input**: Natural conversation in preferred language
- **Text Input**: Type questions directly
- **Image Upload**: OCR for textbook problems and handwritten notes
- Camera capture for instant problem upload

### 4. **Enhanced Learning Experience**
- **Step-by-Step Reasoning**: Detailed explanations with intermediate steps
- **Scaffolded Support**: Progressive hint system
  - Hints revealed one at a time
  - Students can request next hint
  - Encourages problem-solving thinking
- **Conceptual Explanations**: Age-appropriate content
- **Quick Action Buttons**: "Need a Hint", "Step-by-Step", "Show Example"

### 5. **Video Content Integration**
- DIKSHA video integration placeholder
- NCERT-aligned video resources
- Video recommendations based on queries
- Curriculum-verified content

### 6. **Practice Zone**
- Personalized practice questions
- Three difficulty levels: Basic, Intermediate, Advanced
- 5-question practice sessions
- Immediate feedback with explanations
- Hint system for practice questions
